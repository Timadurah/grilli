### Run Locally

To run **Grilli** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/kilogrammie/grilli.git
```

Windows:

```bash
git clone https://github.com/kilogrammie/grilli.git
```
